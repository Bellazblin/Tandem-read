ℹ How to prepare your file
File format:

Option A — Tab-separated (one pair per line):
Ich trinke Kaffee.⟶我喝咖啡。
Sie wohnt in Berlin.⟶她住在柏林。

Option B — Alternating lines:
Ich trinke Kaffee.
我喝咖啡。
Sie wohnt in Berlin.
她住在柏林。

Tips:
• Each sentence needs a translation pair
• Any two languages work
• No special markup needed — you mark practice words during reading
• Lines starting with ===== are ignored
